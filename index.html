📋 Sheets Setup Guide

  1. Open Google Sheets → create a new spreadsheet
  2. Go to Extensions → Apps Script
  3. Paste the script from the box below
  4. Click Deploy → New deployment → Web app
  5. Set access to "Anyone", copy the URL
  6. Paste the URL in the field above
